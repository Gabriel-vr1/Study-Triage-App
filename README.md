# Study Triage

Study Triage is a decision-support tool that helps students prioritise what to study based on academic risk.

Instead of acting like a basic planner, the app calculates an urgency score for each subject using:
- Days until test
- Confidence level
- Current grade
- Credit weighting

Subjects are then ranked using a custom sorting algorithm, allowing students to clearly see what needs attention first.

## Key Features

- Risk-based subject ranking
- Custom urgency score formula
- Manual sorting algorithm (insertion sort)
- Dashboard overview with readiness score
- Deep dive into weak topics
- Semester timeline visualisation
- Local storage (data persists after refresh)
- AI transparency section

## How to Run

1. Download the project
2. Open `index.html`
3. Double-click to run locally in your browser

No setup or installation required.
